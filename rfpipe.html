rfpipe now on ASCL

rfpipe

I am interested in making usable, open software for radio astronomy. My interests in fast radio transient have required developing entirely new approaches to data analysis that had not been demonstrated with traditional software packages.

rfpipe is the next major iteration of that effort. It takes the core functionality of rtpipe and implements it in a more extensible and maintainable way. I’ve also put a lot of effort into making it easy to install (using anaconda), python 2/3 compliant, and well tested. Here are the badges to prove it:

ASCL

If you value open software, please take a look at the Astrophysics Source Code Library. This is a curated index of citable astrophysics software and is designed to reward developers for the time required to write good software. If you make use of rfpipe, please consider citing it or submitting an issue on github.
